# 💪 FitZone Gym Website

## 📖 Description

FitZone Gym Website is a responsive fitness website built using HTML, CSS, and JavaScript. It provides information about gym services, membership plans, trainers, gallery, and contact details while adapting to desktop, tablet, and mobile devices.

## ✨ Features

- Responsive Navigation
- Hero Section
- About Gym
- Services Section
- Membership Plans
- Trainers Section
- Gallery
- Contact Form
- Responsive Design using CSS Media Queries

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Gym-Website/
│
├── index.html
├── style.css
├── script.js
├── images/
└── README.md
```

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open **index.html** in your browser.
